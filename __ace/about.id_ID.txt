Title: Editor Ace
Author: Taufik Nurrohman
URL: http://latitudu.com
Version: 0.0.0

====

> Ganti editor kode sumber bawaan Anda dengan Ace!

**Ace** merupakan editor kode _embeddable_ berbasis JavaScript. Ini memiliki fitur dan performa yang sama persis dengan editor kode asli seperti Sublime, Vim dan TextMate. Ini dapat dengan mudah tertanam dalam setiap halaman web dan aplikasi JavaScript. Ace telah dipertahankan sebagai editor utama untuk [Cloud9 IDE](http://c9.io "Cloud9") dan merupakan penerus dari proyek Mozilla Skywriter (Bespin).

**Referensi:** <http://ace.c9.io>
